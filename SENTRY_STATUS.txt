                              NEXT STEPS (5 min)
═══════════════════════════════════════════════════════════════════════════════

Step 1: SIGN UP (2 min)
  → Go to https://sentry.io
  → Click "Sign Up"
  → Create account with email

Step 2: CREATE PROJECT (2 min)
  → Click "Projects" → "Create Project"
  → Select "Next.js" platform
  → Name it "liftlog-app"

Step 3: GET CREDENTIALS (1 min)
  → Go to Settings → Client Keys (DSN)
  → Copy your DSN
  → (Optional) Get Auth Token from Account Settings

Step 4: CONFIGURE APP (1 min)
  → Create .env.local file
  → Add these lines:
    NEXT_PUBLIC_SENTRY_DSN=your-dsn-here
    SENTRY_ORG=your-org
    SENTRY_PROJECT=liftlog-app

Step 5: TEST (1 min)
  → Run: npm run dev
  → Visit: http://localhost:3000
  → Look for yellow "🔍 Sentry Test Panel"
  → Click "Throw Error (Client)"
  → Check Sentry dashboard → Issues
