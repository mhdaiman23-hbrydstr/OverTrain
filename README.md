# LiftLog
Workout Logging App for Hypertrophy, Strength and Athletics

## AI-Powered Development

This project uses a dual-AI development setup:
- **Claude Code** powered by **GLM 4.6** - Primary coding assistant
- **GLM Scripts** - Code review, testing, and automation tools

### Quick Commands
```bash
npm run dev           # Start development server
npm run glm:verify    # Verify GLM integration
npm run glm:review    # Review staged changes
npm run glm:test      # Generate tests
```
