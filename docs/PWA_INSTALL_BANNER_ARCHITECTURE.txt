================================================================================
                          DETECTION LOGIC FLOW
================================================================================

  App Loads
    │
    ├─→ PWADetection.getEnvironment()
    │   ├─ Check: Is Capacitor wrapper? → environment = "native"
    │   ├─ Check: Is display-mode: standalone? → environment = "standalone"
    │   ├─ Check: Is WebView? → environment = "webview"
    │   └─ Otherwise → environment = "browser"
    │
    └─→ PWADetection.shouldShowInstallPrompt()
        └─ Only returns TRUE if environment === "browser"
            │
            ├─ YES (browser) → Set up beforeinstallprompt listener
            │                  Show banner if event fires
            │
            └─ NO (native/webview/standalone) → Skip setup
                                                  Don't show banner

================================================================================
                          STANDALONE MODE BEHAVIOR
================================================================================

When user installs the app, the manifest.json setting:
  "display": "standalone"

Makes it:
- Open fullscreen (no browser chrome)
- No address bar, tabs, menus
- Appears in app switcher with app icon
- Can be themed with theme_color from manifest
- Can use service worker for offline
- Can access device sensors/hardware

================================================================================
                            KEY BENEFITS
================================================================================

✅ Smart Detection: Banner appears only where it makes sense
✅ No User Confusion: Won't show on webviews where install isn't possible
✅ No Double-Install: Won't show in already-installed native apps
✅ Console Logging: Debug messages help diagnose issues
✅ Zero Configuration: Works out of the box
✅ Standalone Mode: Installed app looks and behaves like native app
✅ Offline Support: Service worker caches for offline use

4. Manifest (public/manifest.json)
